# 项目协作规则

## Language Preference

- 默认使用简体中文与用户沟通。
- 代码标识符、命令、文件路径、日志、API 名称保持原文。

## Project Memory

- 本项目的本地记忆文件是 `.codex/project-memory.md`。
- 每次会话开始处理本项目任务前，先阅读 `.codex/project-memory.md`，再做规划或修改。
- 每次完成本项目任务后，必须把本次任务记录追加到 `.codex/project-memory.md` 的 `任务记录` 部分。
- 任务记录至少包含：日期时间、用户目标、实际修改、涉及文件、验证方式、遗留风险或下一步。
- 如果任务只做调研或分析，没有改文件，也要记录“未修改文件”和关键结论。
- 不要把真实 API key、token、密码、截图中的密钥或完整敏感配置写入项目记忆。
- 除非用户明确同意，不要为了“了解项目”而整仓读取；优先读取 `HANDOVER.md` 和项目记忆文件，必须全仓扫描时先向用户确认。
