# 02 必要模块对标总结（能力地图）

> 本文在「01 行业头部系统特点调研」基础上，将 6 类头部系统的能力求并集，收敛为**广告行业 AI 设计生产与交付系统的 16 个标准模块**。每个模块给出：定义、职责、关键功能点、行业能力等级、本系统现状、缺口等级。

## 1. 模块逐一说明（定义 / 职责 / 关键功能 / 对标）

### M1 用户认证与身份
- **定义**：用户注册、登录、会话、账号安全。
- **关键功能**：邮箱/手机验证、密码找回、登录保护（验证码/限流）、SSO（企业微信/钉钉）、账号注销、操作审计绑定。
- **对标**：Canva/稿定 SSO+审计；Figma 企业 SSO。
- **本系统**：已有 JWT access/refresh、HttpOnly Cookie、token 黑名单、登出。缺验证/找回/SSO/注销。

### M2 权限与组织（RBAC + 团队）
- **定义**：角色、字段级权限、团队工作区、审批、操作审计。
- **关键功能**：多角色（老板/设计师/工人 + 自定义）、**字段级权限**（老板独有预算/定金/余额/客户联系方式）、团队工作区（多成员席位）、审批流、操作日志审计。
- **对标**：稿定内容权限管控+审批；Canva 角色权限+SSO；Figma Dev Mode 注释权限。
- **本系统**：已有三角形色 + 角色倍率（服务端持久化）。**字段级权限（老板独有字段）在开发中**；缺团队工作区、审批流、操作审计日志。

### M3 积分与计费
- **定义**：虚拟积分账户 + 商业化计费。
- **关键功能**：积分账户（冻结/消费/退款/流水）、**订阅套餐、支付充值（微信/支付宝/对账）、按量包、发票、账务对账**。
- **对标**：Canva 订阅+credit；Fotor 按 token；MJ 积分。
- **本系统**：积分账户完整（事务化冻结/扣减/退回 + 流水 + 角色倍率）。缺真实支付、订阅、发票、对账。

### M4 品牌资产库（Brand Kit / DAM）
- **定义**：企业/客户的品牌资产与可商用素材管理。
- **关键功能**：品牌套件（色板/字体/Logo/语调）、客户 VI 库、**正版可商用素材库（图/字体/模板）**、素材标签与检索、授权与版权溯源。
- **对标**：Canva Brand Kit/Brand Voice；稿定品牌规范；创客贴品牌工具箱。
- **本系统**：无。缺。

### M5 模板与提示词库
- **定义**：模板生态与提示词/工作流资产。
- **关键功能**：模板库（分类/封面/可编辑源文件）、**源文件一键转模板**、提示词库、工作流库、模板市集/分享、日更热点模板。
- **对标**：Canva 60 万模板；稿定源文件转模板；创客贴工作流。
- **本系统**：后台模板管理（CRUD）已有；缺模板市集、源文件转模板、提示词社区、分类检索。

### M6 AI 生图与编辑（统一适配层）
- **定义**：统一的多 provider 异步生图 + 生成式编辑。
- **关键功能**：**生图 API 统一适配层（gpt-image-2/万相/可灵/FLUX 抽象为同一异步任务）**、局部重绘 inpainting、扩图 outpainting、智能抠图、风格迁移、参考图一致性/主体库、**视频生成/图生视频**、提示词社区。
- **对标**：fal.ai 模型聚合+异步+webhook；可灵主体库；Firefly 图层编辑；MJ oref。
- **本系统**：gpt-image-2 异步（BullMQ）已有；缺适配层、重绘/扩图/抠图、一致性、视频、社区。

### M7 设计工作台（画布/协同/插件）
- **定义**：可编辑画布 + 协同 + 扩展生态。
- **关键功能**：Fabric/类 Fabric 画布、**实时协同（光标/选区/在线状态）**、组件系统/Design Token、版本分支（branch/对比/合并）、**插件/MCP 扩展架构（cowart 经验）**、AI 原生编辑（标注→生成新图）、评论批注、性能（Web Worker 缩略图、视口裁剪）、**统一滚动条/UI Design Token**。
- **对标**：Figma 协同+组件；tldraw 嵌入式；Penpot MCP；cowart 双画布+MCP。
- **本系统**：Fabric 双画布（生产+创意）雏形 + OCR 旁路。缺协同、组件/Token、分支、MCP、批注、性能修复（当前 4 个 UI 问题）。

### M8 批量生产与多渠道适配
- **定义**：模板套版批量出图 + 多尺寸/多渠道自动衍生。
- **关键功能**：批量渲染引擎（1→N）、数据源/表格批量生成、多版位尺寸自动适配（横/竖/方 10+）、批量导出。
- **对标**：鹿班每秒 8000 张；即创 10+ 广告位；Canva Bulk Create/Autofill。
- **本系统**：无。缺。

### M9 智能抠图/消除/扩图工具箱
- **定义**：生图后的标准后处理算子集合。
- **关键功能**：AI 抠图（去背景）、智能消除（瑕疵/路人）、扩图（outpainting 补画幅）、超分。
- **对标**：五家设计平台均具备；fal FLUX Fill。
- **本系统**：无（仅 sharp 合成）。缺。

### M10 合规与审核
- **定义**：生成/上传内容的合规机审。
- **关键功能**：NSFW/敏感内容审核、**广告法黑词/行业资质拦截**、审核前置到生成链路、一键过审、审核报告。
- **对标**：妙思生成即机审+黑词；Firefly 内容凭证；fal safety_checker。
- **本系统**：OCR 旁路仅做文字校验；缺 NSFW、黑词、机审前置。

### M11 报表与分析（含投放回流）
- **定义**：经营多维分析 + 投放效果归因。
- **关键功能**：公司收支报表（周/月/季/年/自定义）、素材级效果分析、A/B 优选、**投放数据回流归因**（对接各平台 Marketing API）。
- **对标**：阿里妈妈素材体检；巨量创意数据 API。
- **本系统**：老板端报表（周/月/季/年/自定义 + AI 集成）开发中；缺投放回流、素材级归因、A/B。

### M12 行业热点与灵感
- **定义**：行业热点爬取与趋势洞察。
- **对标**：即梦灵感热榜；巨量爆款捕手。
- **本系统**：老板端「行业热点🔥日频率」开发中（抖音/小红书/微博爬取 + 西安本地资讯，Node 运行时）。

### M13 订单与履约
- **定义**：实物交付的核价/预飞/工单/物流闭环。
- **关键功能**：在线报价核价引擎（材质×尺寸×工艺）、印前预飞校验（出血/CMYK/300DPI/字体/总墨量）、生产工单状态机、材质工艺库、**客户审阅确认（留痕）**、物流交付、开票对账、工厂 ERP/MES 对接。
- **对标**：印刷家自助报价+智能检档；包小盒报价+预飞；阳光印网 OMS。
- **本系统**：仅本地 orders.json（Node 写入 + sanitize 防损）。缺核价、预飞、工单、审阅、物流、对账。

### M14 开放平台与 API
- **定义**：对外 REST API + Webhook + SDK + 嵌入。
- **关键功能**：开放 API（生图/合成/导出/模板/资产）、OAuth2 授权、Webhook（任务完成回调）、SDK、Embed/iframe 嵌入、用量计量与配额。
- **对标**：Canva Connect；万相营造 API（MCP）；巨量 Marketing API。
- **本系统**：无对外 API。缺。

### M15 监控运维与告警
- **定义**：可观测性与稳定性保障。
- **关键功能**：指标采集（任务失败率、队列积压、网关可用率、积分异常波动）、日志聚合、告警（阈值+通道）、链路追踪、健康检查扩展。
- **对标**：行业标配 Prometheus/Grafana + Sentry。
- **本系统**：仅 `/health` 健康检查；缺指标、告警、日志聚合。

### M16 协作与审批流
- **定义**：多人协作与审批。
- **关键功能**：画布内评论批注、@人、审批流（导出前 BOSS 复核）、成员席位管理、离职交接。
- **对标**：稿定审批流；Figma 注释；Canva 审批。
- **本系统**：无。缺。

## 3. 优先级结论（与 03 缺口清单对应）

**P0（不做不算同类产品，必须补）**：M2 字段级权限落地、M4 品牌资产库、M5 模板市集+转模板、M6 生图适配层+重绘/扩图、M7 协同+MCP+UI 性能、M10 合规机审、M13 核价+预飞+工单、M15 监控、M14 开放 API、M3 支付订阅。

**P1（重要差异化）**：M8 批量生产、M9 抠图工具箱、M11 投放回流、M16 协作审批、M1 验证/找回/SSO。

**P2（后续迭代）**：M12 热点深化、M4 版权素材采购、M6 视频生成、M13 物流/ERP 对接。
