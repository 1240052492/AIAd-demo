# 01 行业头部系统特点调研

> 调研时间：2026-07-16
> 调研方法：6 个研究 Agent 并行，覆盖 6 类头部系统，基于 2024–2026 公开资料（官网、文档、行业报道、云市场 API 说明）。
> 目的：为「AdCraft AI（神笔）」广告行业 AI 设计生产与交付系统，建立行业能力基线，支撑必要模块收敛与缺口分析。

## 1. 类别 A：AI 在线设计平台（中小商家/营销/电商向）

### 1.1 代表系统与定位
- **Canva（可画）**：全球最大「模板+AI（Magic Studio）」一体化视觉内容平台，月活 2.4 亿+。
- **稿定设计**：面向国内中小企业/门店的「商业设计+批量化+品牌合规管控」平台，企业 500 万+。
- **创客贴**：「AI 设计智能体+全场景打印+企业内容中台」，1 亿+用户、20 万+企业。
- **图怪兽**：专注电商卖家「模板批量化出图+多平台尺寸适配+一键抠图」的轻量工具。
- **Fotor**：技术驱动、以「AI 修图/抠图/生成」见长的全球化工具，6 亿+用户、2025 上线开发者 API。

### 1.2 核心能力矩阵
- **模板库生态**：Canva 60 万+、稿定 30 万+、创客贴 100 万+原创模板，覆盖 200+ 场景，日更追热点。
- **品牌套件 / Brand Kit**：色板、字体、Logo、语调（Brand Voice），Enterprise 无限套；生成默认套用 VI。
- **一键 AI 生成**：Magic Design（自然语言→多版式）、创客贴「直接生成可分层设计稿」（背景层/商品层/文案层）。
- **批量套版 / 批量出图**：源文件转模板、表格/AI 批量生成，1 分钟 1→N；Canva Bulk Create / Autofill API。
- **素材库 / 版权图**：1 亿级正版可商用素材+可商用字体，规避侵权。
- **智能抠图 / 消除 / 扩图**：五家均具备，电商商品图秒级批量。
- **AI 文案**：Magic Write、稿定智能文案 20 类、创客贴接入 DeepSeek 润色+11 国翻译。
- **团队协作与权限**：实时协作+审批流+角色权限+SSO；稿定内容权限管控、离职交接。
- **导出与印刷出口**：PNG/JPG/PDF；创客贴/稿定/图怪兽打通「在线印刷」一件起订。
- **计费**：订阅（Canva Pro $12.99/月、Teams $14.99/人/月）+ credit（Fotor $8.99/月起按 token）。
- **开放 API**：Canva Connect（Design/Export/Asset/Autofill/Brand Template/Comment，REST+OpenAPI+SCIM+审计）；Fotor 开发者 API。

### 1.3 头部共性特征
- 模板规模+版权素材构成供给护城河；AI 把「设计门槛」降到自然语言。
- Brand Kit 是「品牌一致性」的核心载体，也是 B 端付费支点。
- 批量化（Bulk Create / Autofill）把设计产能放大 10–100 倍，直接服务电商 SKU。
- 开放 API 把设计能力嵌入客户自有系统（ERP/门店），形成生态粘性。

### 1.4 对本系统的启示（详见 03 缺失清单）
明显缺失：品牌套件、模板市集与源文件转模板、批量套版/多尺寸适配、智能抠图/消除/扩图工具箱、正版素材版权库、团队协作席位+审批流、订阅计费、对外开放 API。

## 2. 类别 B：AI 创意生成平台（文生图/图生视频）

### 2.1 代表系统与定位
- **即梦 Dreamina（字节）**：抖音/剪映生态一站式 AI 图文视频，强中文语义与「创作-分发」闭环。
- **通义万相（阿里）**：Wan 系列多模态，文/图生图+生视频+音画同步，开放 DashScope API。
- **Adobe Firefly**：训练数据全授权+公共领域，商业版权无忧，深度嵌入 Creative Cloud。
- **Midjourney**：艺术质感与一致性标杆（V7 + Omni Reference + Draft 对话模式），无官方 API。
- **可灵 AI（快手）**：视频一致性领先，主体库（ID 漂移<0.03）+ 原生音频，按秒计费 API。
- **海螺 AI（MiniMax）**：电影级运镜与物理仿真，主体参考+导演模式，有 API。
- **fal.ai / Replicate**：聚合 FLUX/SD 的推理 API 平台，按图/兆像素计费，强 SDK + webhook 异步。

### 2.2 核心能力矩阵
- **文生图 / 图生图 / 局部重绘（inpainting）/ 扩图（outpainting）**：主流全覆盖。
- **智能抠图 / 风格迁移 / 参考图一致性**：可灵主体库、MJ oref/sref、万相多图参考。
- **视频 / 图生视频**：即梦/万相/可灵/海螺原生；Firefly beta。
- **对话式编辑**：Firefly 文本指令视频、MJ Draft、即梦智能画布。
- **提示词社区**：即梦「一键同款」、MJ Gallery、海螺 Agent 模板。
- **API / SDK**：即梦/万相/可灵/海螺 REST+SDK；fal OpenAI 兼容端点。
- **计费/额度**：订阅积分（MJ/Firefly/即梦）、按秒（可灵）、按兆像素（fal $0.025/MP）。

### 2.3 头部共性特征
- **异步化是标配**：通义万相 `X-DashScope-Async` 提交 task_id→轮询；fal `subscribe`+`queue`(webhook)。与本项目 BullMQ+gpt-image-2 架构同构。
- **一致性做成产品能力**：跨图/跨镜头「不变脸」是头部标配（主体库 / Omni Reference）。
- **版权无忧是 B 端卖点**：Firefly 仅授权数据 + C2PA 内容凭证 + IP 赔偿，是商业安全唯一标杆。
- **编辑算子化**：inpainting / outpainting / 抠图作为「生图后的标准后处理算子」独立成服务。

### 2.4 对本系统的启示
明显缺失：局部重绘/扩图/outpainting、参考图一致性/主体库、视频生成、生图 API 统一适配层（多 provider 抽象）、提示词社区、额度包/订阅、版权水印+C2PA、NSFW 内容审核。

## 3. 类别 C：企业级智能设计与广告生产（大厂出品）

### 3.1 代表系统与定位
- **阿里鹿班 / 阿里妈妈创意中心**：淘系官方 AI 电商设计，双 11 单日 10 亿海报；「洞察-制作-投放」一站式。
- **京东羚珑**：泛零售商家设计伙伴，强在电商主图/视频/3D 批量与多尺寸适配。
- **腾讯广告妙思**：基于混元大模型，「创意生成—机审—直连投放」全链路。
- **巨量创意 / 即创（字节）**：抖音电商一站式智能创意生产，文生视频/数字人/商品图文批量。
- **百度擎舵**：多模态 AIGC，2 分钟 100+ 条文案、3 分钟数字人视频。

### 3.2 核心能力矩阵
- **批量模板出图**：鹿班多版本 A/B、羚珑批量白底/透底、即创批量商品卡。
- **商品图自动生成**：背景合成+商品库对接（妙思、即创 AI 试穿）。
- **多尺寸/多渠道自动适配**：1 张主图衍生横/竖/方 10+ 广告位（即创、妙思 LORA 衍生）。
- **品牌规范合规**：妙思审核前置到 prompt/底图每一步，过审时效降 80%；羚珑企业专区统一视觉。
- **AIGC 文案**：阿里妈妈随热点优化、擎舵 100+ 条/2 分钟。
- **投放效果回流**：阿里妈妈「素材体检」+ 投放数据反哺生成；妙思基于素材偏好优化 CTR。
- **API / 开放平台**：羚珑企业 API、妙思开发者 API、腾讯广告 Marketing API。

### 3.3 头部共性特征
- **规模化是根基**：鹿班「每秒 8000 张」渲染；批量渲染引擎 = 模板套版 + 参数化批量。
- **商业平台闭环**：生成→审核→投放→效果回流内嵌，素材直接进投放账户。
- **合规即能力**：审核前置 + 虚拟人像免授权，把「过审」做成产品特性。
- **降本可量化**：素材生产从「天」缩到「分钟」，CTR 翻番。

### 3.4 对本系统的启示
明显缺失：批量渲染引擎、多渠道尺寸自适应引擎、品牌规范引擎、投放效果回流分析、与电商/媒体平台打通、企业级开放 API + 素材合规审核。

## 4. 类别 D：设计协作工作台（画布/协同/插件与 MCP）

### 4.1 代表系统与定位
- **Figma**：浏览器原生 WebGL/WebGPU 云端多人实时协同，设计协作事实标准。
- **tldraw**：开源 React 无限画布 SDK，可把生产级白板/协同嵌入自有产品。
- **MasterGo（莫高设计）**：国产产设研一体化，设计工程化 + AI 整页生成。
- **即时 AI（即时设计）**：协作版 Sketch，自然语言生成可编辑 UI，插件广场。
- **Penpot**：开源，首个把 MCP Server 暴露给 LLM 的设计工具。
- **cowart（参考）**：基于 tldraw 的双画布（生产画布+创意画布）+ MCP 适配层 + skill 剧本。

### 4.2 核心能力矩阵
- **画布渲染**：Figma C++→WASM + 瓦片化→WebGPU；tldraw DOM 渲染；Penpot SVG/Rust-WASM。
- **图层系统**：层级化图层+布尔运算+智能路径；tldraw shape record + 嵌套 transform。
- **实时协同**：Figma 中心服务器为真相源 + 属性级 LWW；tldraw sync（Durable Objects + WebSocket）；即时/MasterGo 50 人协作。
- **版本历史/分支**：Figma 分支（branch，review/merge）；Penpot 版本历史。
- **组件与 Design Token**：Figma Variables/Styles、MasterGo 变量+插槽、Penpot Design Tokens 直接映射 CSS。
- **插件/扩展**：Figma/Penpot 沙箱插件 + API/Webhook；tldraw 自定义 shape/tool；Penpot 额外提供 MCP Server。
- **AI 原生**：Figma AI 生成 UI；cowart「标注→生成干净新图放原图旁（before/after 版本链）」而非覆盖。
- **导出**：SVG/PNG/PDF/代码；Canva Magic Switch 多平台一键改写。
- **权限/批注**：RBAC + 链接权限 + 画布内评论 @人；Figma Dev Mode 注释。
- **性能**：瓦片/视口裁剪、WASM、Web Worker 异步缩略图、增量同步、CDN。

### 4.3 cowart 双画布 + MCP 经验（重点）
- 生产画布（交付）+ 创意画布（探索）分离，降低「生成覆盖原稿」的损失。
- 把高频易错动作收敛为**少量 MCP 窄工具**（如 `insert_generated_image`、`get_selection`），让 agent 读结构化 JSON 而非截图。
- 选区每 250ms 写小 JSON（`cowart-selection.json`）作为 agent 上下文模板。
- 红色箭头标注 → 生成干净新图放原图旁，形成 before/after 版本链。

### 4.4 对本系统的启示
对应：已有 Fabric 双画布雏形 + gpt-image-2 异步 + sharp 合成 + OCR 旁路。
明显缺失：实时协同、组件系统/Design Token、版本分支、插件/MCP 扩展架构、AI 原生编辑、评论批注、性能与缩略图优化（当前前端 4 个 UI 问题：滚动条、缩略图损坏、AI 需求分析显示等）、统一滚动条/UI 一致性（Design Token 规范化）。

## 5. 类别 E：广告投放与创意生产管线（Ad Tech 中台）

### 5.1 代表系统与定位
- **巨量引擎（即创/巨量创意）**：字节一站式 AIGC 营销创意，抖音–千川闭环直连。
- **腾讯广告（妙思/Marketing API）**：混元大模型，「生成即机审即投放」。
- **阿里妈妈（万相台/创意中心）**：电商 AI 创意+全域投放中台，实时反馈。
- **百度营销（擎舵/基木鱼）**：擎舵 2.0 多模态 + 基木鱼智能建站 + 免费 Marketing AI API。
- **快手磁力（磁力开创/创意）**：可灵大模型短视频创意 + 女娲数字人 + 磁力智投。

### 5.2 核心能力矩阵
- **创意生产**：文/图/视频/数字人；即创数字人 2.0、擎舵 1000+ 形象、妙思 AI 口播。
- **多版位适配**：即创 1 次上传自动生成 10+ 广告位尺寸；妙思 LORA 一图衍生 N 版位。
- **A/B 与优选**：妙思多变体对比；即创「爆款裂变」拆解重组。
- **素材审核**：妙思生成即机审 + 一键过审，派生素材继承结果走机审免人审（分钟级）；内置黑词/合规拦截。
- **投放回流与归因**：巨量创意数据 API 提供素材级最细粒度报表（归因/成本/展示/点击/转化）。
- **程序化批量**：擎舵单次最高 100 张、模板制图 400 张。
- **素材标签**：妙思 AI 标签 + 二创追溯；巨量属性标签化。
- **开放 API**：巨量 Marketing API（OAuth2 + SPI 订阅 + SDK）、妙思 API、万相营造 API（云市场按次 + MCP）、百度免费 Marketing AI API。

### 5.3 头部共性特征
- **闭环数据飞轮**：生产→投放→优选→再生产，投放结果反哺生成模型与推荐（爆款捕手）。
- **规模与合规**：即创 618 素材消耗 +81%；妙思「过审确定性」把审核前置。
- **开放**：Marketing API 把创意能力以 REST+OAuth+SPI 对外，避免轮询。

### 5.4 对本系统的启示
明显缺失：素材标签/分类体系、A/B 优选、机审合规引擎+黑词库、投放数据回流+效果归因报表、程序化批量派生、数字人/视频生成、灵感热榜/爆款捕手、开放 Marketing API。

## 6. 类别 F：印刷/实物交付与订单履约

### 6.1 代表系统与定位
- **印刷家 / 印通天下**：面向印刷厂的「印刷电商+印前自动化+ERP」一体化 SaaS（自助报价、智能检档、拼版、生产管控、财务）。
- **阳光印网（爱阳光）**：企业非核心物资一站式采购，PNP 智能 SaaS 撮合 1.5 万供应商，覆盖印刷/包装/喷绘等 16 大类。
- **包小盒（包小印）**：在线 3D 包装设计+小批量印刷电商，2D/3D 同步、一键自动报价。
- **易企秀 / 稿定 / 创客贴**：AIGC 出图后「在线印刷出口」直达合作工厂。
- **广告标识工厂**：门店招牌/喷绘/UV/亚克力/发光字源头工厂，按文件+材质人工核价。

### 6.2 核心能力矩阵
- **在线报价/核价**：按纸张/尺寸/工艺自动算成本；包小盒「一秒出报价」；标识厂「公分×单价」公式。
- **订单管理（OMS）**：下单-设计-制作-审批-物流-仓储-结算全流程；多渠道收款与权限。
- **印前文件校验**：包小盒强制 CMYK、文字转曲、图片嵌入、≥300DPI、出血、含结构矢量；印通智能检档+转 PDF/X。
- **文件预飞（Preflight）**：出血 3mm、安全区 3–5mm、字体嵌入/转曲、总墨量 TAC≤300%、叠印/陷印；标准 PDF/X-1a、FOGRA39/GRACoL。
- **生产工单流转**：ERP 生产管控、排产与延期预警。
- **材质/工艺库**：白卡/瓦楞/铜版 + 烫金/UV/凹凸；标识厂 UV 平板、亚克力、发光字、雪弗板。
- **进度追踪状态机**：待生产/制作中/待发货/已签收。
- **客户确认/审阅**：下单前反复核对设计稿、留痕。
- **物流交付 + 开票对账**：对接顺丰/德邦；送货单自动生成 + 电子回签 + 对账单。
- **API/ERP 对接**：紫云/印通深度 ERP；阳光印网 PNP 对接客户采购系统。

### 6.3 头部共性特征
- **履约可靠**：印通年处理 1200 万笔订单；阳光 7×24 跟单 + 3 天退换。
- **报价透明**：参数化公式即时出价、拼单降本。
- **供应链网络**：阳光 1.5 万厂商 + 本地化产能。

### 6.4 对本系统的启示
对应：我们的「工厂施工参考图」「矢量交付（SVG/PDF）」≈ 标识厂安装图/CDR/PLT 结构图；sharp 合成 + Fabric 编辑 ≈ 包小盒 2D/3D 同步预览。
明显缺失：在线报价核价引擎、印前预飞校验、生产工单状态机、材质工艺库、客户审阅确认、物流交付、开票对账、工厂 ERP/MES 对接。

## 7. 跨类别共性结论（收敛到必要模块，见 02）

将 6 类头部系统的能力求并集，广告行业 AI 设计生产与交付系统的**完整能力域**收敛为 16 个标准模块（详见 02）。其中本系统当前仅具备约 40%（认证/RBAC 基础、积分、项目/资产/版本、AI brief、异步生图、环境合成、Fabric 画布、导出、后台、OCR 旁路），其余为缺口（详见 03）。

**行业确定的「必做题」（不做就不算同类产品）**：
1. 品牌套件 + 品牌模板库（品牌一致性）
2. 批量套版 + 多尺寸/多渠道自动适配（产能）
3. 智能抠图/消除/扩图工具箱（后处理标准件）
4. 生图 API 统一适配层（多 provider 异步队列）
5. 素材合规机审（NSFW + 广告法黑词）
6. 实时协同 + 评论批注（协作）
7. 在线核价 + 印前预飞 + 生产工单（实物交付闭环）
8. 开放 API 平台（生态与嵌入）
9. 监控运维告警（稳定性）
10. 订阅计费 + 支付充值（商业化）
